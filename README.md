# Homebrew Tap for aws-tui

This is the official [Homebrew](https://brew.sh/) tap for [aws-tui](https://github.com/aws-tui/aws-tui).

## Installation

```bash
brew tap aws-tui/tap
brew install aws-tui
```

## Upgrading

```bash
brew upgrade aws-tui
```

## Uninstallation

```bash
brew uninstall aws-tui
brew untap aws-tui/tap
```

## Formulas

| Formula | Description |
|---------|-------------|
| [aws-tui](Formula/aws-tui.rb) | Terminal User Interface for AWS services |
